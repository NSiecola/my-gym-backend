# API para App de Academia

Este é o repositório do backend para um aplicativo simples de registro de treinos de academia (workout tracker).

## Objetivo do Projeto

O principal objetivo deste projeto é **educacional**. Foi desenvolvido como um exercício prático para aprender e aprofundar os conhecimentos em desenvolvimento backend, focando em:

* **Criação de uma API RESTful** utilizando as API Routes do Next.js.
* **Modelagem de um banco de dados** relacional com PostgreSQL para uma aplicação real.
* **Manipulação de dados** com comandos SQL (INSERT, SELECT, etc.).
* **Conexão segura** entre a aplicação e o banco de dados.

O frontend da aplicação será desenvolvido separadamente, consumindo esta API.

## Tecnologias Utilizadas

* **Backend:** Next.js (API Routes)
* **Linguagem:** TypeScript
* **Banco de Dados:** PostgreSQL (hospedado na Neon)
* **Driver do Banco:** `node-postgres` (pacote `pg`)
